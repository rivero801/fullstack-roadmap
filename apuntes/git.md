# Git

## git init

Inicializa un repositorio de Git para nuestro proyecto.

## git status

Verifica cuáles son aquellos archivos que estuvimos modificando y nos indica cuáles son.

## git add .

Agrega todos los archivos modificados a Git.

## git commit

Es el encargado de subir los archivos que indicamos en el add. O sea subir una nueva versión.

## git push

Sube las modificaciones a GitHub.

## git pull

Es el encargado de descargar los cambios del repositorio remoto.

## git clone

Es el encargado de clonar un repositorio ya existente.

## git branch

Muestra o crea ramas.

## git checkout

Cambia de rama.

## git merge

Fusiona ramas.
